Cambios incluidos en esta versión:
- Ventas:
  - historial más compacto tipo tarjetas
  - filtros prolijos por texto, día, mes, año y rango
  - editar, eliminar y compartir ticket imagen
  - corrección del guardado de edición
- Gastos:
  - historial con filtros
  - editar y eliminar
  - exportar PDF resumen con detalle mes a mes
- Caja:
  - historial con editar y eliminar
- Stock:
  - se quitó el botón superior "Nuevo producto"
  - quedan editar / eliminar en el listado
  - sigue exportación PDF por categorías
- Cuentas corrientes:
  - botones Imagen / PDF / Eliminar sólo visibles cuando elegís una cuenta
  - movimientos con editar / eliminar
  - cargos vinculados a ventas se corrigen desde la venta
  - clientes nuevos con código correlativo tipo 0001, 0002, etc.
- Presupuestos:
  - cliente cargado con buscador
  - armado de ítems más compacto
  - botón único "+" para agregar ítems
  - estados Enviado / Aprobado en la lista
  - vista previa con acciones visibles sólo al seleccionar presupuesto
- Usuarios:
  - editar y eliminar
- Ajustes:
  - nueva pantalla con tema día / noche
- Responsive:
  - mejoras de adaptación para smartphone

Nota:
- La exportación de lista PDF de productos sigue siendo EXPORTAR, no importar desde un PDF externo.
