Instalá MundoLED Control en este dispositivo.

MundoLED Control

Panel interno para caja, stock, gastos y cuentas corrientes.

Primer acceso:

1. Cargá tu configuración de Firebase en firebase-config.js.

2. En Firebase Authentication habilitá Email/Password.

3. Creá manualmente el primer usuario desde Firebase Authentication.

4. En Firestore, creá el documento users/UID con: { role: "admin", active: true, name: "Tu nombre" }.
