CONFIGURACION NECESARIA EN FIREBASE

1) Authentication
- Entrar a Firebase > Authentication > Sign-in method
- Activar: Anonymous

2) Firestore Database
- Crear la base de datos si todavia no existe

3) Reglas de Firestore
Pegar estas reglas:

rules_version = '2';
service cloud.firestore {
  match /databases/{database}/documents {
    match /{document=**} {
      allow read, write: if request.auth != null;
    }
  }
}

4) Primer ingreso
- Si la coleccion users esta vacia, el primer ingreso en modo Administracion crea el primer admin automaticamente.
- Si ya probaste antes y quedo algo mal, elimina la coleccion users y volve a ingresar.

5) Datos sugeridos para primer ingreso
usuario: admin
contrasena: 1234

IMPORTANTE
- Si sigue fallando, revisar la consola del navegador (F12) y buscar errores de Firebase.
